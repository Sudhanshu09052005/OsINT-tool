TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

INSTALLATION ISSUES:

Q: "ModuleNotFoundError" when running
A: pip install -r requirements.txt
   pip install --upgrade pip

Q: Python not found
A: Install from https://www.python.org
   Add to PATH (check during installation)
   Restart terminal

Q: Permission denied on macOS/Linux
A: sudo pip install -r requirements.txt
   OR: pip install --user -r requirements.txt

SCANNING ISSUES:

Q: "Connection Timeout"
A: Check internet connection
   Verify target is accessible
   Try different target
   Check firewall settings

Q: "WHOIS Data Unavailable"
A: Some domains have private registration
   Other data (DNS, SSL) usually available
   Try different domain

Q: Subdomain scan returns few results
A: Tool uses common subdomain list
   Larger lists can be added for more results
   Not all subdomains are publicly accessible

EXPORT ISSUES:

Q: PDF export fails
A: Check write permissions
   Ensure disk space available
   Try different save location
   Close files that might be locked

Q: JSON export blank
A: Run scan first
   Check data was gathered
   Try different filename

🔴 PROBLEM 2: Browser Mode Open Nahi Ho Raha
═══════════════════════════════════════════════════════════════════════

CHECK 1: Flask Installed Hai?
──────────────────────────────
Command Prompt में run करें:
    python -c "import flask; print('Flask OK')"

Agar error aaye:
    pip install flask

CHECK 2: Port 5000 Free Hai?
─────────────────────────────
Agar "Port already in use" error aaye:
1. Edit करें: web_app.py
2. Last line change करें:
   FROM: app.run(debug=False, host='0.0.0.0', port=5000)
   TO:   app.run(debug=False, host='0.0.0.0', port=5001)
3. Browser में जाएं: http://localhost:5001

CHECK 3: Templates Folder Hai?
──────────────────────────────
Run करें: python setup_dirs.py
Yeh templates/ aur reports/ folder bana dega

🔴 PROBLEM 3: Desktop Mode Open Nahi Ho Raha
═══════════════════════════════════════════════════════════════════════

CHECK 1: Tkinter Available Hai?
────────────────────────────────
Command Prompt में run करें:
    python -c "import tkinter; print('Tkinter OK')"

Agar error aaye:
• Windows: Python reinstall करें, "tcl/tk" option check करें
• Linux: sudo apt-get install python3-tk

CHECK 2: Enhanced Tool File Hai?
─────────────────────────────────
Check करें file exist karti hai:
    enhanced_osint_tool.py

Agar nahi hai, toh project incomplete hai.

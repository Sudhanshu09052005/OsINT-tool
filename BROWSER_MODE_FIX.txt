❌ COMMON ERRORS AUR SOLUTIONS:
═══════════════════════════════════════════════════════════════════════

ERROR 1: "No module named 'flask'"
───────────────────────────────────
SOLUTION:
    pip install flask

ERROR 2: "Address already in use" / "Port 5000 busy"
─────────────────────────────────────────────────────
SOLUTION A - Different Port Use Karein:
    python web_app_simple.py
    (Automatically port 5001 use karega)

SOLUTION B - Manual Port Change:
    Edit web_app.py
    Last line change to:
    app.run(debug=False, host='0.0.0.0', port=5001)

ERROR 3: "Template not found"
──────────────────────────────
SOLUTION:
    python test_browser_mode.py
    (Automatically template create kar dega)

ERROR 4: Browser khulta nahi
─────────────────────────────
SOLUTION:
1. Check terminal mein "Running on http://..." dikha?
2. Manually browser mein jao:
   • http://localhost:5000
   • http://127.0.0.1:5000
   • http://localhost:5001 (agar port change kiya)

ERROR 5: "Import Error" ya "Module not found"
───────────────────────────────────────────────
SOLUTION:
    python web_app_simple.py
    (Yeh minimal dependencies use karta hai)

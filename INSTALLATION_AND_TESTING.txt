📦 STEP-BY-STEP INSTALLATION
═══════════════════════════════════════════════════════════════════════════

STEP 1: Verify Python Installation
───────────────────────────────────
Open Command Prompt and run:

    python --version

You should see: Python 3.7 or higher

If not installed:
- Download from: https://www.python.org/downloads/
- During installation, CHECK "Add Python to PATH"
- Restart Command Prompt after installation

STEP 2: Navigate to Project Folder
───────────────────────────────────
In Command Prompt:

    cd "c:\Users\niran\OneDrive\Desktop\New folder (3)"

Or simply open Command Prompt in this folder:
- Hold Shift + Right-click in folder
- Select "Open command window here" or "Open PowerShell window here"

STEP 3: Install Dependencies
─────────────────────────────
Run this command:

    pip install -r requirements.txt

This will install all required packages:
✓ Flask (web framework)
✓ ReportLab (PDF generation)
✓ BeautifulSoup (web scraping)
✓ dnspython (DNS queries)
✓ python-whois (WHOIS lookups)
✓ validators (URL validation)
✓ And more...

Wait for installation to complete (may take 2-5 minutes).

STEP 4: Create Required Directories
────────────────────────────────────
Run:

    python setup_dirs.py

This creates:
- templates/ (for web interface)
- reports/ (for exported files)

STEP 5: Verify Installation
────────────────────────────
Run this test command:

    python -c "import flask, reportlab, bs4, dns, whois; print('✓ All modules installed successfully!')"

If you see the success message, you're ready!

🔧 TROUBLESHOOTING TESTS
═══════════════════════════════════════════════════════════════════════════

PROBLEM: "Module not found" error
SOLUTION:
    pip install -r requirements.txt --upgrade

PROBLEM: "Port 5000 already in use"
SOLUTION:
    Method 1: Close other applications using port 5000
    Method 2: Edit web_app.py, line 133, change:
        app.run(debug=False, host='0.0.0.0', port=5001)

PROBLEM: Tkinter error in Desktop mode
SOLUTION:
    Windows: Reinstall Python with Tkinter checked
    Linux: sudo apt-get install python3-tk

PROBLEM: Scan fails or times out
SOLUTION:
    - Check internet connection
    - Try with http://example.com
    - Increase timeout in advanced_scanner.py

PROBLEM: PDF generation fails
SOLUTION:
    pip install --upgrade reportlab Pillow

PROBLEM: "Permission denied" when installing
SOLUTION:
    pip install -r requirements.txt --user

❓ QUICK ANSWERS
════════════════════════════════════════════════════════════════════════════════

Q: I just want to use it now!
A: Read QUICKSTART.txt (5 min), then double-click run.bat (Windows)
   or run: pip install -r requirements.txt && python osint_tool.py

Q: What does this tool do?
A: Gathers detailed information about websites (WHOIS, DNS, IP, SSL, etc.)
   and generates professional PDF reports.

Q: How long does a scan take?
A: Typically 2-5 minutes depending on website size and your connection.

Q: What will I find?
A: Domain registration info, DNS records, IP address, SSL certificate details,
   technologies used, subdomains, email addresses, and more.

Q: Is this legal?
A: Yes, if you scan websites you own or have explicit permission to scan.
   See ethical guidelines in README.md

Q: What are the system requirements?
A: Python 3.7+, 200MB disk space, 200MB RAM, internet connection.
   See INSTALLATION.md for details.

Q: Can I use this on production?
A: For authorized security testing only. Always get written permission first.

Q: How do I export results?
A: Click "Export to PDF" or "Export to JSON" button after scanning.

Q: What if something breaks?
A: Check INSTALLATION.md troubleshooting section.
   Reinstall dependencies: pip install -r requirements.txt

Q: Can I modify the tool?
A: Yes! See LEARNING_GUIDE.md for customization examples.

Q: Where's the source code?
A: Everything is in osint_tool.py (well-commented, 1000+ lines).


🆘 EMERGENCY HELP
════════════════════════════════════════════════════════════════════════════════

PROBLEM: "Python not found"
SOLUTION: Install Python from https://www.python.org
          Make sure to check "Add to PATH"

PROBLEM: "ModuleNotFoundError"
SOLUTION: pip install -r requirements.txt
          pip install --upgrade pip

PROBLEM: "Connection Timeout"
SOLUTION: Check internet connection
          Try different domain
          Check firewall/proxy settings

PROBLEM: "Tool won't start"
SOLUTION: python osint_tool.py (from terminal)
          Check Python version: python --version
          Reinstall dependencies

PROBLEM: "Report doesn't generate"
SOLUTION: Check write permissions
          Try different save location
          Ensure disk space available

PROBLEM: "I can't find the files"
SOLUTION: They're in: C:\Users\niran\OneDrive\Desktop\New folder (3)
          Open folder in Windows Explorer
